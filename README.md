# Download Manager

A beautiful Material You download manager browser extension that replaces your new tab page with a clean, modern interface for managing downloads.

## Features

- Clean Material You design
- View and manage all downloads in one place
- Replace new tab page with download interface
- Modern UI with Tailwind CSS

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension is now installed!

## Usage

- Click the extension icon to open the download manager
- Or simply open a new tab to see your downloads

## Permissions

- `downloads`: Access to browser download history
- `storage`: Store extension settings
- `downloads.open`: Open downloaded files

## Technologies

- Manifest V3
- Tailwind CSS
- Material Symbols
- JavaScript
